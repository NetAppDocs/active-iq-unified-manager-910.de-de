= Was RBAC ist
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
RBAC (rollenbasierte Zugriffssteuerung) bietet die Möglichkeit, zu steuern, wer Zugriff auf verschiedene Funktionen und Ressourcen im Active IQ Unified Manager Server hat.
